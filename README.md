# Documentation 
## initialisation de dépot

```bash
git init
git remote add origin SSH_Repo
git status
git add README.md
git commit
git branch -M main
git push origin main
git log
git show
git restore --staged Readme.md /*permet de restorer au précédente étape*/
```
